# cursor-setup-project
Learning project documenting Cursor, AI tools, and GitHub setup workflow
## Overview

This repository documents the installation and setup process for Cursor IDE, AI coding tools, Git, and GitHub integration.

The purpose of this project was to learn and complete a technical setup workflow while documenting the process and troubleshooting any issues encountered.

## Tools Installed

- Cursor IDE
- Claude Code extension
- Codex extension
- Git for Windows
- GitHub account and public repository

## Steps Completed

1. Created a GitHub account and public repository
2. Installed Cursor IDE
3. Connected GitHub to Cursor
4. Installed Git for Windows
5. Configured Git during installation
6. Opened Cursor and enabled terminal access
7. Cloned the GitHub repository using Git commands
8. Opened and edited the repository inside Cursor

## Issues Encountered and Solutions

### Git command not recognized

Issue:
The terminal returned:

`git is not recognized`

Solution:
Git was not installed on the system. I downloaded and installed Git for Windows and configured PATH settings to allow Git commands to run inside Cursor.

### Repository cloning setup

Issue:
The Git clone workflow was initially unclear because Git functionality was unavailable.

Solution:
After installing Git and restarting Cursor, I successfully cloned the repository using the terminal and connected the local project with GitHub.

## Key Learning

This project helped me understand how Cursor, Git, and GitHub work together, as well as the importance of troubleshooting setup and development environments independently.
